# Health Care Advisor

A simple static health care advisor app built with HTML, CSS, and JavaScript.

## Features

- Personalized wellness advice based on age, gender, activity level, and goals
- Responsive layout for desktop and mobile
- Clean and modern UI

## Usage

1. Open `index.html` in your browser.
2. Enter your age, gender, activity level, and health goal.
3. Click **Get Advice** to see tailored recommendations.

## Files

- `index.html` - main page markup
- `styles.css` - page styling
- `script.js` - form handling and advice logic
